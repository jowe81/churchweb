Music Database Syntax

Basic Search

Just start typing a song title or scripture reference. When the desired song gets suggested, select it.

Advanced Search

You can search by writer, theme, or lyrics fragment if you choose the respective field

More Advanced Search

You can use keys of the form -key=value to further restrict your results

Search by musical key

  • type -key=guitar-friendly to limit results to pieces that have at least one arrangement that is considered guitar-friendly
  • type -key=G to get results that have at least one arrangement in the key of G
  • type -key=nochange to get results with at least one arrangement without keychange

Search by musical instrument

  • type -instruments=violin to get results that include at least one arrangement with a violin score
  • type -instruments=leadsheet to get results with a lead sheet
  • type -instruments=chordsheet to get results with a chord sheet
  • type -instruments=4part to get results with 4-part vocals

Search by musical style

Provided the database administrator is using these tags, you can

  • type -style=hymn to get results that include at least one hymn arrangement
  • type -style=contemporary to get results with at least one contemporary arrangement
